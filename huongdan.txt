* Cụ thể cài đặt:

Bước 1 — Cài dependencies
->  pip install -r requirements.txt
Bước 2 — Tạo file .env và thêm API key
-> GROQ_API_KEY=gsk_xxxxxxxxxxxxxxxxxxxxxxxx
Bước 3 — Đặt file dữ liệu CSV vào thư mục data/
-> Các file dataset có đủ
Bước 4 — Build index (chỉ chạy lần đầu)
-> python setup.py
Bước 5 — Chạy ứng dụng
-> python -m streamlit run main.py
